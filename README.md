## Laravel e-commerce example application

### Run this project

In the project directory, you can run:

```shell
composer install
```

```shell
php artisan migrate --seed
```

```shell
php artisan serve
```

Open [http://localhost:8000](http://localhost:8000) to view it in the browser.
